# FROST-Meltybrain-PCB
Repository for all of the PCB design files in KiCAD for the FROST meltybrain

## COMPONENTS
- H3LIS331DL Sparkfun breakout board
- Arduino Nano microcontroller
- WS2812B RGB LEDs
